# Cpp_BenchMark_Lab

A C++ benchmarking and algorithm optimization laboratory.

## Goals

- Implement classic algorithms
- Benchmark execution time
- Compare optimized versions
- Study memory usage and cache behavior
- Build production-quality C++ engineering practices

## Project Structure

```
src/algorithms  Algorithm implementations
include/        Header files
src/benchmark   Benchmark code(main.cpp)
docs/           Documentation
scripts/        Utility Python scripts
src/utils       Utility implementations
```
